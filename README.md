# DigiCodes — Tienda de Códigos Digitales

Sitio web estático para la venta de códigos digitales (Google Play, iTunes/Apple, PSN, Xbox, Steam, Amazon, Netflix, Spotify, Free Fire, Roblox y más).

## Características
- Catálogo de productos con **precios referenciales tipo G2A**.
- Filtros por categoría (Google, Apple, PSN, Xbox, Steam, Otros).
- **Métodos de pago:** efectivo y transferencia bancaria.
- **Contacto por WhatsApp:** +503 7016 4013 (con mensaje precargado por producto).
- Diseño moderno, responsive y con botón flotante de WhatsApp.

## Archivos
- `index.html` — estructura de la página.
- `styles.css` — estilos y diseño responsive.
- `script.js` — catálogo de productos, filtros y enlaces de WhatsApp.

## Cómo usar
1. Abre `index.html` en tu navegador, o publícalo con **GitHub Pages**.
2. Edita el arreglo `products` en `script.js` para cambiar productos y precios.
3. El número de WhatsApp se configura en la constante `WA_NUMBER` dentro de `script.js`.

## Personalización
- Cambia colores en las variables `:root` de `styles.css`.
- Agrega o quita productos en `script.js`.

> Los precios son referenciales y pueden variar. Consulta disponibilidad por WhatsApp.
